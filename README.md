# 🗓️ Bookington

Sistema de gerenciamento de reservas desenvolvido para centralizar, organizar e otimizar o processo de agendamento em estabelecimentos que trabalham com reservas.

---

## 🚀 Objetivo e Funcionalidades

O objetivo deste projeto é desenvolver uma solução digital para gerenciamento de reservas, permitindo o controle centralizado das informações, a redução de retrabalho e a melhoria da comunicação entre os responsáveis pelo processo. A plataforma busca substituir processos manuais por um sistema mais organizado, eficiente e intuitivo, permitindo:
- Cadastro de clientes e funcionários;
- Autenticação por login e senha;
- Controle de acesso por perfil de usuário;
- Solicitação, cadastro e gerenciamento de reservas;
- Confirmação e cancelamento de reservas;
- Consulta de horários disponíveis;
- Visualização e edição de reservas;
- Controle centralizado das informações.
